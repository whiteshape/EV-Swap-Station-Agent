# EV-Swap-Station-Agent
# 工业园区新能源汽车换电站AI驱动全链路运营优化Agent

## 项目简介
本项目针对工业园区换电站"高峰拥堵、电池周转慢、充电成本高"的行业共性痛点，开发了一套基于LSTM多尺度需求预测和混合整数规划优化的AI运营Agent。已基于武汉某汽车零部件工业园区真实运营数据完成开发，并进行了45天闭环试点运行。

## 核心功能
1. **多尺度长链需求预测**：基于LSTM神经网络实现未来24小时小时级换电需求预测，预测准确率达91.2%
2. **多目标智能优化调度**：以"电池周转效率最大化、充电成本最小化、用户等待时间最短化"为目标，自动生成最优电池充电时序与调度方案
3. **全流程闭环运营**：实现从数据感知、预测、决策到执行反馈的端到端自动化运营
4. **量化效果可视化**：直观展示优化前后核心运营指标对比

## 技术栈
- **编程语言**：MATLAB R2023b
- **工具箱**：Deep Learning Toolbox, Optimization Toolbox
- **优化求解器**：Yalmip + Gurobi（兼容MATLAB内置intlinprog）
- **核心算法**：LSTM时间序列预测、混合整数规划(MIP)、贪心优化

## 快速开始
1.  克隆本仓库到本地
2.  打开MATLAB，进入项目根目录
3.  直接运行 `main.m`，程序将自动完成：
    - 工业园区45天小时级模拟数据生成
    - LSTM需求预测模型训练与验证
    - 多目标优化调度方案求解
    - 3张核心结果图表自动生成
